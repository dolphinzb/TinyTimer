# add-about-page: 关于页面

## 概述

添加一个"关于"页面，展示 TinyTimer 计时器应用的产品介绍和版本信息。用户可以通过设置页面点击"关于"进入该页面。

## 问题与动机

当前应用缺少专门的产品介绍页面，用户无法方便地查看应用信息和功能介绍。添加关于页面可以：
- 提供专业的应用介绍
- 展示版本信息方便问题反馈
- 提升产品的专业形象

## 预期结果

用户点击 设置 → 关于 后，进入关于页面，页面包含：
- 应用名称和图标
- 版本号（动态读取自 BuildConfig）
- 产品功能介绍
- 核心特性说明

## 影响范围

- 新增页面：`AboutPage.kt`
- 修改 `SettingsPage.kt`：添加导航回调
- 修改 `MainActivity.kt`：添加导航路由

## 附加信息

产品版本号：动态读取自 BuildConfig.VERSION_NAME
