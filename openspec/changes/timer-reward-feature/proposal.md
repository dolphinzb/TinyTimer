## Why

计时奖励功能通过在计时结束时展示分组排名和奖励，提升用户的使用动机和成就感。当用户在某个分组上投入大量时间时，排名和奖励的展示能够有效增强粘性，让计时过程更加有趣。

## What Changes

- **排名计算与展示**: 计时结束时，在**当前分组内**计算当次计时时长排名（统计该分组内有多少条记录比当次更短），显示第1-3名
- **庆祝动效**: 排名前三名（用时最少的前三名）时弹出庆祝动效：第1名显示烟火，第2-3名显示鲜花
- **奖励展示**: 依据名次展示对应的奖品图片和名称，以半透明弹窗形式呈现
- **奖品配置**: 复用现有 `PrizeEntity` 奖品系统，按分组+名次关联奖品

## Capabilities

### New Capabilities

- `timer-ranking`: 计时排名能力，包含排名计算逻辑和排名展示 UI
- `reward-display`: 奖励展示能力，包含奖品配置管理、弹窗展示和庆祝动效

### Modified Capabilities

- 无（现有计时功能仅做 UI 扩展，不改变核心计时逻辑）

## Impact

- **UI 层**: TimerPage 扩展计时结束后的排名/奖励弹窗
- **数据层**: 复用现有 `PrizeEntity` + `PrizeDao` 奖品系统，无需新增数据结构
- **动画**: 使用 Compose Canvas 自绘烟火/鲜花粒子动效
